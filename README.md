# dotfiles

## Prerequisites

- [Nix package manager](https://nixos.org/)
- Nix flakes
- [Homebrew](https://brew.sh/)

## Install

```console
$ ./install.sh
```

Manual steps:

- iTerm2 preferences: General > Preferences > Load preferences from a custom folder or URL > ./config/iterm2
- System Preferences: Keyboard > Input Sources > Edit > Spelling > Set Upâ€¦ > Add "U.S. English" and drag to top
